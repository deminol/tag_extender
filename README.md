# Tag extender

python script extract ID3 tags from media files and convert them to XML

usage: tag_extender.py [-h] [-i INPUT] [-d FOLDER] [-o OUTPUT]

Tag extender for XML-feed

optional arguments:

  -h, --help            show this help message and exit

  -i INPUT, --input INPUT
                        input file path

  -d FOLDER, --folder FOLDER
                        input folder

  -o OUTPUT, --output OUTPUT
                        output xml file

## Requirements

For install requiremnts:

* cd to the directory where __requirements.txt__ is located
* run: ```pip install -r requirements.txt``` in your shell
